FIFA World Cup 2026 Predictor League — Figma Design System & UI Prompt

🎨 Design Brief

Create a modern, playful, animated website/app design for the FIFA World Cup 2026 Predictor League — a fantasy football prediction platform. The design must be responsive (desktop + mobile) using a component-based system compatible with React Native rendering. Think FPL meets FIFA+ with World Cup 2026 energy.

🌍 Brand Identity

App Name: WC26 Predictor League

Tagline: "Predict. Compete. Win."

Personality: Sporty, Modern, Competitive, Fun, Premium

Inspiration References:

- FPL (Fantasy Premier League) — league/gameweek structure, points system UI

- FIFA+ App — World Cup branding, match cards

- Sorare — modern sports fantasy UI, card-based design

- OneFootball — clean match cards, live score interface

- Bet365/FanDuel — live score interfaces (without gambling elements)

- Apple Sports — minimal, elegant sports data display

Card Design:

- Left: colored icon circle (40px)

- Center: title (bold) + description + timestamp

- Right: unread dot (accent-teal, 8px) or action button

- Swipe left to dismiss (mobile)

- Slide-in animation from right on arrival

9. Analytics / Stats Page

Sections:

- Prediction Accuracy — Donut chart (correct vs incorrect, by type)

- Points Over Time — Line chart (points per matchday)

- Streak Indicator — 🔥 flame icon with number (e.g., "🔥 3 correct in a row")

- Comparison — Your accuracy vs league average (bar chart)

- Best Predictions — List of highest-scoring predictions

- Category Breakdown — Score/Team/Player accuracy percentages

Visual Style:

- Minimal, clean charts with smooth curves

- Animated on scroll into view (draw-in effect)

- Color-coded: Score predictions (teal), Team (blue), Player (gold)

- Dark mode friendly (no white backgrounds on charts)

✨ Animation Specifications

Page Transitions:

- Between pages: Shared element transitions (fixture card → prediction page)

- Modal/Sheet: Slide up from bottom with spring physics (mobile)

- Tab switch: Cross-fade with 200ms duration

- Navigation: Push from right (forward), push from left (back)

🌙 Theme Modes

Dark Mode (Default — Recommended):

- Background: #0D0D1A (near black with blue tint)

- Cards: #1A1A2E with 1px border #2A2A4E

- Text: #FFFFFF (primary) / #A0A0B0 (secondary)

- Accent elements have subtle glow/neon effect

- Shadows: colored (e.g., teal glow under active elements)

- Charts: bright colors on dark backgrounds

Light Mode:

- Background: #F5F5F7 (warm grey)

- Cards: #FFFFFF with subtle shadow (0 2px 8px rgba(0,0,0,0.08))

- Text: #1B0A3E (primary) / `#
